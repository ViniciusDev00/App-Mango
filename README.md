# Aplicação de Filmes e Séries

## Descrição do Projeto
Esta é uma aplicação inovadora para gerenciar e visualizar uma ampla gama de filmes e séries. A aplicação oferece uma interface amigável, permitindo aos usuários buscar, filtrar e salvar suas produções favoritas.

## Recursos Principais
- **Catálogo de Filmes e Séries**: Navegue por categorias e descubra novas produções.
- **Pesquisa Avançada**: Encontre títulos rapidamente usando filtros como ano, gênero e avaliação.
- **Listas Favoritas**: Salve seus filmes e séries favoritos para fácil acesso.
- **Avaliações e Críticas**: Deixe suas opiniões e veja o que outros usuários estão dizendo.

## Tecnologias Utilizadas
- **Frontend**: React ou Angular (escolher um)
- **Backend**: Node.js com Express
- **Banco de Dados**: MongoDB
- **Autenticação**: JWT (JSON Web Tokens)

## Como Executar o Projeto
1. Clone o repositório:
   ```bash
   git clone https://github.com/ViniciusDev00/App-Mango.git
   ```
2. Navegue até o diretório da aplicação:
   ```bash
   cd App-Mango
   ```
3. Instale as dependências:
   ```bash
   npm install
   ```
4. Execute a aplicação:
   ```bash
   npm start
   ```
